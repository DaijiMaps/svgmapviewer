- How to generate .json

cd packages/demos/Test/src/data
inkscape map.svg

Extensions -> Daiji Maps -> Address: Resolve: Resolve shops...

This generates:

- origin.json
- bbox.json
- viewbox.json
- floors.json
- floors-addresses-Main.json
- floors-names-Main.json
- facilities.json
  in packages/demos/Test/src/data
